# Kontiva Official Brand Asset Manifest

This package contains the approved Kontiva logo and brand assets.

The original design concept is locked and must remain intact.

## Approved Visual Concept

Preserve:

- dark rounded-square app icon,
- white stylized `K`,
- white horizontal top and bottom bars,
- red minus/accent mark,
- clean Kontiva wordmark,
- small red square accent in the wordmark,
- red accent `#E11D2E`,
- serious, professional, elegant, Swiss-compatible tone.

## Included Asset Types

This package may include:

- PNG app icons,
- PNG wordmarks,
- PNG concept sheet,
- PDF brand reference,
- Markdown brand brief,
- SVG/vector masters where supported.

## Primary Source Assets

Use these as source for app packaging:

- `kontiva-icon-1024.png`
- `kontiva-icon-512.png`
- `kontiva-logo-concept-sheet.png`
- `kontiva-wordmark.png`
- `kontiva-wordmark-dark.png`

Use SVG files as vector masters where supported.

## Not Allowed

- No redesign.
- No recolor.
- No stretching.
- No cropping.
- No replacing with generic finance icons.
- No mascot.
- No childish visual language.
- No canton coat-of-arms usage.
