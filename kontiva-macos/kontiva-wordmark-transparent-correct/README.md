# Kontiva transparent wordmark package

Corrected package based on the approved Kontiva wordmark assets.

## Included versions

- `kontiva-wordmark-dark-on-light-transparent`
  - Charcoal wordmark for light backgrounds.
  - Uses `#121A22` and the red accent `#E11D2E`.
  - Transparent background.

- `kontiva-wordmark-light-on-dark-transparent`
  - Off-white wordmark for dark backgrounds.
  - Uses `#F6F7F8` and the red accent `#E11D2E`.
  - Transparent background.

## Formats

- `svg/` — editable SVG masters.
- `png/` — transparent PNG exports in 512, 1080, and 2160 px widths.
- `preview/` — preview sheet showing the assets on light and dark surfaces.

## Usage

Use the dark/charcoal version on white, warm-light, or other light UI surfaces.

Use the light/off-white version on charcoal, black, or other dark UI surfaces.

Do not recolor, stretch, add shadows, place on busy backgrounds, add canton emblems, add flags, or modify the mark/wordmark relationship.
