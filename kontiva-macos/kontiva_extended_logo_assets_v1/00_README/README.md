# Kontiva Extended Logo Asset Package

This package extends the approved Kontiva logo direction into practical app, website, and developer handoff assets.

## Structure

- `01_brand_masters/` contains the untouched approved source files.
- `02_app_icons/` contains generated app icon exports for macOS and Windows.
- `03_web_favicons/` contains favicon and web manifest assets.
- `04_wordmarks/` contains resized wordmark PNGs for light and dark contexts.
- `05_social_preview/` contains Open Graph preview images.
- `06_claude_upload_friendly/` contains PNG/PDF/Markdown files suitable for Claude project uploads.
- `07_developer_notes/` contains usage rules and implementation notes.

## Brand Constants

- Product name: Kontiva
- Charcoal: `#121A22`
- Off-white: `#F6F7F8`
- Warm light: `#F8F7F4`
- Swiss red accent: `#E11D2E`
- Soft border: `#DCE1E5`

## Important

Do not redesign, recolor, stretch, distort, or regenerate the Kontiva mark. Generated assets in this package are derived from the approved PNG sources.
