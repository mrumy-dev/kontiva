# Kontiva Brand Brief for Claude

Kontiva is a serious Swiss personal finance desktop app for budgeting, bills, salary, private documents, and tax organization.

## Approved Logo Direction

Use the attached PNG concept sheet as the visual source of truth.

The approved mark is:
- a geometric `K` monogram,
- framed by top and bottom ledger/accounting balance lines,
- paired with a small Swiss red accent,
- placed inside a dark premium rounded-square app tile for app/desktop usage,
- paired with a clean modern wordmark for website and software UI usage.

## Product Tone

Kontiva should feel like a trustworthy Swiss finance tool, not a playful budget app.

Use words and UI patterns that communicate:
- clarity,
- control,
- privacy,
- structure,
- calm,
- financial confidence.

## Visual Rules

Colors:
- Charcoal: `#121A22`
- Off-white: `#F6F7F8`
- Warm light: `#F8F7F4`
- Swiss red accent: `#E11D2E`
- Soft border: `#DCE1E5`

Use the red accent sparingly. It should help recognition, not dominate the design.

## Asset Usage

Use:
- `kontiva-icon-1024.png` for app icon direction.
- `kontiva-wordmark.png` for light website/header contexts.
- `kontiva-wordmark-dark.png` for dark contexts.
- `kontiva-logo-concept-sheet.png` for the complete approved brand direction.

SVG production masters exist in the separate full brand asset ZIP, but Claude project uploads should use PNG/PDF/Markdown alternatives.
