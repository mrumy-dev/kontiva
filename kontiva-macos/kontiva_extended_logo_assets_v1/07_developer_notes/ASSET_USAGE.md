# Kontiva Asset Usage

## macOS App

Use:

- `02_app_icons/macos/KontivaAppIcon.appiconset/`

In Xcode, add this folder to the asset catalog or copy the PNGs into an existing `AppIcon.appiconset`.

The generated set includes:

- 16x16 at 1x and 2x
- 32x32 at 1x and 2x
- 128x128 at 1x and 2x
- 256x256 at 1x and 2x
- 512x512 at 1x and 2x

## Windows App

Use:

- `02_app_icons/windows/kontiva.ico`

This contains multiple embedded icon sizes.

## Website

Use:

- `03_web_favicons/favicon-16x16.png`
- `03_web_favicons/favicon-32x32.png`
- `03_web_favicons/apple-touch-icon.png`
- `03_web_favicons/android-chrome-192x192.png`
- `03_web_favicons/android-chrome-512x512.png`
- `03_web_favicons/site.webmanifest`

Use the wordmarks from:

- `04_wordmarks/light/` for light backgrounds.
- `04_wordmarks/dark/` for dark backgrounds.

Use the Open Graph preview images from:

- `05_social_preview/`

## Claude Project Uploads

If Claude project files reject SVGs, upload files from:

- `06_claude_upload_friendly/`

Recommended:

- `kontiva-logo-concept-sheet.png`
- `kontiva-brand-reference.pdf`
- `kontiva-brand-brief.md`
- `kontiva-icon-1024.png`
- `kontiva-wordmark.png`
- `kontiva-wordmark-dark.png`

## Rules

- Preserve the original files in `01_brand_masters/`.
- Do not use canton emblems, coats of arms, flags, crests, piggy banks, dollar signs, coins, mascots, or generic bank/security logos.
- Use the red accent sparingly.
- Keep Kontiva serious, calm, premium, private, and Swiss-compatible.
