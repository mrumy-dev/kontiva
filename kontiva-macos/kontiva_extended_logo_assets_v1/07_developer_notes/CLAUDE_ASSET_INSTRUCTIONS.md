# Instructions For Claude Code

Use the provided Kontiva brand assets exactly as supplied.

## Required Behavior

- Inspect the asset folder before building UI.
- Use `kontiva-icon-1024.png` as the source for app icon generation.
- Use `kontiva-wordmark.png` on light UI backgrounds.
- Use `kontiva-wordmark-dark.png` on dark UI backgrounds.
- Use the concept sheet and brand reference as visual truth.
- Keep untouched brand masters separate from generated app assets.

## Forbidden

- Do not redesign the logo.
- Do not recolor the logo.
- Do not distort the logo.
- Do not crop the logo in a way that changes its balance.
- Do not replace the logo with generic finance, bank, shield, piggy-bank, coin, chart, or dollar-sign imagery.
- Do not use canton coats of arms, flags, crests, or emblems.

## Brand Tone

Kontiva should feel like a serious Swiss private finance desktop app:

- calm
- precise
- trustworthy
- local-first
- private
- professional
