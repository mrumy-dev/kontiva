# Kontiva Premium Profile Icons v2

This package contains a new premium profile icon system for Kontiva profile creation.

The previous icon direction was rejected and is not reused here.

## Contents

- `svg/` - 16 SVG masters
- `png-256/` - 16 PNG exports at 256x256
- `png-512/` - 16 PNG exports at 512x512

## Icon Groups

- 8 abstract human profile icons
- 4 household/couple/family profile icons
- 4 monogram-style icons

## Style Direction

These icons are intentionally restrained:

- serious
- minimal
- premium
- native macOS-compatible
- suitable for a privacy-first finance app
- closer to account identity tiles than generic avatar illustrations

## Brand Palette

- Charcoal: `#121A22`
- Off-white: `#F6F7F8`
- Warm light: `#F8F7F4`
- Swiss red accent: `#E11D2E`
- Soft border: `#DCE1E5`

## Rules

- Do not use these as app logos.
- Do not place the Kontiva logo inside profile icons.
- Do not add flags, canton emblems, crests, or coats of arms.
- Do not replace these with cartoon faces or realistic portraits.
- Use the red accent sparingly.
